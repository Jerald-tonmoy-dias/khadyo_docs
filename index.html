Introduction

Khadyo Restaurant Software is one of the promising softwares to manage a restaurant in a very easy way. We have strong believe that it is going to be one of the most popular script in upcoming days.

The Installation processes will be shown in this documentation with written article and also with video references. An user will be able to install the system after buying it from codecanyon without any coding knowledge. We have provided written and Video guideline of most of the features of the system.

Requirements

8+ in your server is required to install khadyo restaurant software.

  1. OpenSSL PHP Extension
  2. PDO PHP Extension
  3. Mbstring PHP Extension
  4. Tokenizer PHP Extension
  5. XML PHP Extension
  6. Ctype PHP Extension
  7. JSON PHP Extension
  8. ZIP PHP Extension
  9. BCMath PHP Extension

Please contact your server provider if you get any error regarding these while the installation process.

Most Important

Khadyo restaurant software can only be installed on the root of your domain. For example- www.yourdomain.com It can not be installed in the subdirectory of your current domain. For example: www.yourdomain.com/khadyo-software

If you want to install in any subdirectory, feel free to contact us.

Upload File to Server

To upload the downloaded file from codecanyon to your server, follow the steps below.

  1. Login to your cpanel
    • e.g “https://yourdomain.com/cpanel”
  2. Click File Manager
  3. Go to the public_html folder
  4. Upload here the "upload_this.zip" file which is in the downloaded file from “CodeCanyon”
  5. Go back to the public_html folder
  6. "Right click" on the Zip File, you will see some options there
  7. Now, Click on Extract
  8. Confirm Extract Files. The Zip File is now extracted successfully. You are all set to go to the next step
  9. Do not forget to delete "upload_this.zip" file after successful extraction.

Create Database

Create a new database from your Cpanel MySQL database. If you are new to cpanel- Follow the procedure given here to create the database.

  1. Login to your cpanel
    • e.g “https://yourdomain.com/cpanel”
  2. Type MySQL in the Search Box
  3. Select MySQL Databases. A page will come
  4. Type database name in the input field and click Create Database. A new page will come with successful notification Click Go Back
  5. Now create an user for the database. To create an user, scroll down to MySQL Users on the same page
  6. Insert Username & Password, Click Create User. A new page will come with successful notification. Click Go Back
  7. Now Add User to Database so that the user can use the database. Scroll down to Add User to Database
  8. Select the User & Database you just have created
  9. Click on Add. A new page will come
  10. Click on Make Changes
  11. Click on Go Back

The database is now set to install the system.

Server Installation

If you have followed the previous steps accordingly, You are all set to install the system.

  1. Navigate to the link below replacing "yourdomain" with your original domain
    • http://yourdomain.com Or (for SSL) https://yourdomain.com
  2. Click "Install". A page will come. Follow the steps written there
  3. If your php version does not match the requirement of khadyo restaurant software (8+), contact your server provider
  4. If your curl is not enabled, contact your server provider
  5. Now you need to set Database Host, Database Port, Database Name, Database Username, Database Password
    • Database Host - Type “localhost”
    • Database Port - your_port (e.g 3306)
    • Database Name - yourdomain_databasename
    • Database Username - yourdomain_username
    • Password - Type the password of the user you created from MySQL Database
  6. Click Save the configuration
  7. Click on import sql or import demo sql
  8. Name, Email, Phone no, Password to create an Admin
  9. Save and that's all, use your system.

The database is now set to install the system.

Installation on Localhost

Please follow the 4 steps carefully to install the system in your local server.

Step 1 - Copy Files

  1. Create a folder in your local server, name it "khadyo" ( inside htdocs folder if you are using xampp )
  2. Copy and paste here the "upload_this.zip" file which is in the downloaded file from “CodeCanyon”
  3. Right click" on the Zip File
  4. "Extract here" the zip file

Step 2 - Create Database

  1. Start apache server
  2. Create a database in your local database, ( navigate to "localhost/phpmyadmin" if you are using xampp )

Step 3 - Add Virtual Host

  1. The following steps are for xampp server in windows os (operating system). If you are using any other server/os, please configure your server with the same configuration as it is here.
  2. Configure apache server as shown in the video below
  3. These are the lines you need to add in the configuration:
  4. <VirtualHost *:80>
    DocumentRoot "your_khadyo_directory_path"
    ServerName localhost
    <Directory "your_khadyo_directory_path">
    </Directory>
    </VirtualHost>
  5. Restart apache server

Step 4 - Install The System

  1. Navigate to the link below
    • http://localhost
  2. Click "Install". A page will come. Follow the steps written there
  3. You need to set Database Host, Database Port, Database Name, Database Username, Database Password
    • Database Host - Type “localhost”
    • Database Port - your_port (e.g 3306)
    • Database Name - databasename
    • Database Username - username (default "root")
    • Password - type your password ( keep password field blank if you didn't set any password )
  4. Click Save the configuration
  5. Click on import sql or import demo sql
  6. Name, Email, Phone no, Password to create an Admin
  7. Save and that's all, use your system.
